Запуск робота

Запустить документацию

Собрать контейнер: docker build -t armbot .

Запустить контейнер: docker run -d --name docs -p 80:80 docs


Запустить робота

1 терминал: roslaunch armbot_move armbot.rviz.launch (модель)

2 терминал: roslaunch armbot_move move.rviz.launch (подписчик на движения)

3 терминал: ./scripts/run.sh start или ./scripts/run.sh start docker (публикатор движений)


Описание команд

Файл command_description.txt. Пример описания команды:

direct:0.2872 4.83407e-10
right:-0.180411 -0.223463
left:-0.180411 0.223463

Файл commands.txt. Пример описания перечня выполняемых команд:

right 6
left 4
