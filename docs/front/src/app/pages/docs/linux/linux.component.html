Linux

Удалить папку с содержимым: rm -r devel

Исправить ошибку переда запуском скриптов (преобразование окончаний строк из формата DOS в формат UNIX): sed -i -e 's/\r$//' scripts/run.sh

Установить для вебсокета npm install -g wscat

Отправка вебсокета wscat -c ws://localhost:9080/ws/coordinate, следующая строка - просто 3 числа через пробел.
